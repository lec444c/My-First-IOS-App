# CSTacticsApp

CSTacticsApp is a small SwiftUI iOS sample app for Counter-Strike tactical utility lineups.

The first version contains:

- A map list with Mirage
- A placeholder 2D Mirage tactical map
- 10 static Mirage utility points
- A detail page for each lineup

Open `CSTacticsApp.xcodeproj` in Xcode to run the app.
